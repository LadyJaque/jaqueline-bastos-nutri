📋 Sobre o Projeto

Este é o repositório do site profissional de Jaqueline Bastos Nutricionista, uma plataforma moderna de reeducação alimentar 100% online. O projeto foi desenvolvido com tecnologias de ponta para oferecer uma experiência excepcional aos clientes.

O site apresenta informações sobre os serviços, permite agendamento de consultas via WhatsApp e oferece um formulário de anamnese detalhado para coleta de dados nutricionais dos clientes.

✨ Funcionalidades Principais

📧 Sistema de E-mail Automático

Envio automático de e-mails para confirmação de formulários de contato e anamnese com formatação profissional em HTML.

📝 Formulário de Anamnese

Formulário multi-step completo para coleta de dados nutricionais, histórico de saúde e hábitos alimentares.

💬 Formulário de Contato

Formulário simples para visitantes entrarem em contato com confirmação automática por e-mail.

📱 Design Responsivo

Interface moderna e responsiva que funciona perfeitamente em desktop, tablet e dispositivos móveis.

🎨 Design Premium

Design "Sage Editorial" com minimalismo escandinavo e luxo wellness, cores sofisticadas e tipografia elegante.

🔐 Autenticação OAuth

Sistema de autenticação seguro com Manus OAuth integrado para gerenciamento de usuários.

🛠️ Stack Tecnológico

O projeto utiliza um stack moderno e robusto para garantir performance, segurança e escalabilidade:

Frontend React 19 + TypeScript
Styling Tailwind CSS 4 + OKLCH Colors
Backend Express 4 + Node.js
API tRPC 11 (Type-safe RPC)
Database MySQL/TiDB + Drizzle ORM
Authentication Manus OAuth + JWT
Email Resend / SendGrid API
Testing Vitest + React Testing Library

📧 Sistema de E-mail Automático

O projeto implementa um sistema robusto de envio de e-mails que integra automaticamente com os formulários do site.

Provedores Suportados

  • Resend - Recomendado para projetos modernos
  • SendGrid - Provedor robusto e confiável
  • Gmail - Via Nodemailer com senha de app

Funcionalidades

✉️ E-mail de Anamnese

Envia a anamnese completa formatada em HTML para o cliente com todos os dados preenchidos.

✉️ E-mail de Contato

Confirmação automática de mensagens de contato com formatação profissional.

🔄 Tratamento de Erros

Sistema robusto com tratamento de erros e logging detalhado de todas as operações.

Configuração

Para usar o sistema de e-mail, configure as seguintes variáveis de ambiente:

EMAIL_PROVIDER=resend EMAIL_API_KEY=sua_chave_api_aqui EMAIL_FROM_ADDRESS=noreply@nutrijaquebastos.com

✅ Testes Unitários

O projeto inclui testes unitários completos com Vitest para garantir a qualidade do código:

  • 7 testes automatizados, todos passando ✓
  • Cobertura de formatação de e-mail em HTML
  • Validação de tratamento de erros
  • Testes de integração com tRPC

Execute os testes com: pnpm test

🚀 Como Usar

Instalação

# Clonar o repositório git clone https://github.com/seu-usuario/jaqueline-bastos-nutri.git cd jaqueline-bastos-nutri # Instalar dependências pnpm install # Configurar variáveis de ambiente cp .env.example .env.local # Executar migrations do banco de dados pnpm db:push # Iniciar servidor de desenvolvimento pnpm dev

Executar Testes

pnpm test

Build para Produção

pnpm build
